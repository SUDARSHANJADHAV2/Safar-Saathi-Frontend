# Safar Saathi - Frontend

Safar Saathi is a modern travel booking web application frontend built using React and Vite.  
It integrates with a secure Spring Boot backend to provide authentication, trip booking, and payment functionality.

---

## 🚀 Tech Stack

- React (Vite)
- JavaScript (ES6+)
- Axios (API integration)
- Tailwind CSS
- React Router
- Context API (Authentication)
- JWT-based login system

---

## 🔐 Features

- User Registration & Login
- Role-Based Dashboard (Admin / Customer / Vendor)
- Trip & Package Browsing
- Booking Management
- Payment Integration
- Secure Route Protection
- Responsive UI

---

## 📂 Project Structure

src/
- components/
- pages/
- routes/
- context/
- api/
- layouts/

---

## ⚙️ How To Run

1. Clone repository:
   git clone <repo-link>

2. Install dependencies:
   npm install

3. Start development server:
   npm run dev

Application runs on:
http://localhost:5173

---

## 🔗 Backend Repository

This frontend connects to the Safar Saathi Spring Boot backend.

Make sure backend is running before starting frontend.
